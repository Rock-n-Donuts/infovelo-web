Code de conduite

La politesse et l'amabilité sont de mise envers les cyclistes, personnels des travaux publics, automobilistes, policiers, piétons, citoyens... L'équipe du projet se garde le droit de modifier ou censurer tout contenu ne respectant pas le code de conduite.

Projet bénévole

La plateforme est loin d'être parfaite, nous le savons. Nous croyons aussi que c'est un pas dans la bonne direction. Merci de votre indulgence.

L'aide supplémentaire, les commentaires constructifs, les idées sont les bienvenues écrivez-nous à : rock_n_donuts@gmail.com

Mission

Donner l'information sur l'état des pistes afin qu’elle soit empruntée.

Vision
